# Mini Groupware Workflow

Spring Boot, JSP, Mybatis, MariaDB를 활용한 미니 그룹웨어 업무 요청 관리 프로젝트입니다.
코덱스는 업무에 따라 변경하며 사용하지만 주로 gpt5.4 mini 모델을 사용합니다.

## 프로젝트 목적
홈페이지/그룹웨어 유지보수 및 구축 및
팀 기반 업무 요청과 처리 흐름을 직접 구현하며 Spring MVC 구조와 DB 연동 흐름을 학습하는 것이 목표입니다.
스프링 부트 기반의 프로젝트를 코덱스를 이용해 학습 및 복습이 목표입니다.
(코덱스는 도움을 흐름과 방향에 도움을 주고 저는 이를 이해하며 사용합니다
 단순 코드 복붙이 아닌 흐름과 구조를 이해하는데 목표를 두고 있습니다)

 ## 주요기능
 - 팀 등록/조회
 - 회원 등록/조회
 - 업무 요청 등록/조회
 - 업무 요청 상태 변경
 - 처리 이력 등록

 ## 학습 목표

 - Controller-Service-Mapper 구조 이해
 - JSP 기반 화면 처리
 - MyBatis와 MariaDB 연동
 - RESTful URL 설계
 - JUnit 5, Mockito, MockMvc 등 테스트 적용
 - GitHub 기반 개발 기록 관리
 - 코덱스를 이용한 바이브코딩 학습
